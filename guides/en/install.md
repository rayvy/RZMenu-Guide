# Installing RZMenu

This page is a starter example. Replace it with your real installation steps.

## 1. Check the Blender version

Use the Blender version that the current RZMenu build targets. If the add-on is made for Blender 5.0, do not mix it with another version without testing.

## 2. Copy the add-on

The add-on folder should be placed in the Blender user add-ons directory:

```text
Blender Foundation/Blender/5.0/scripts/addons/RZMenu
```

## 3. Enable the add-on

Open Blender, go to Add-ons, and enable RZMenu.

## 4. Check the interface

After enabling it, the RZMenu panels and commands should appear. If they do not, restart Blender and check the console for errors.

## 5. Run a test

Before real work, import a small test asset and confirm that export creates the expected files.
