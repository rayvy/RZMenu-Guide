# Code Examples

Use this tab for small examples, module patterns, configs, and copy-pasteable pieces.

## Example format

```py
def example_module_entry(context):
    print("RZMenu module example")
```

## Suggested page structure

- What problem this snippet solves.
- What files it touches.
- Full snippet.
- Expected result.
- Common ways to break it.

> Keep examples small. A giant script dump belongs in the asset library, not inside a tutorial page.
