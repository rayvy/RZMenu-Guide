# Assets

This tab can work as a clean file dump for RZMenu-related materials.

You can store:

- `.rmzct` files
- `.rzm` files
- example modules
- themes
- test scenes
- screenshots and previews

The visible cards below are generated from:

```text
asset-dump/catalog.json
```

Files live in:

```text
asset-dump/files/
```

Previews live in:

```text
asset-dump/previews/
```
