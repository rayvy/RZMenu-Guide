# Installation and Dependencies

This page covers the usual first-launch problems: missing packages, missing editor buttons, and the annoying restart loop.

## Why does the first install sometimes need a second restart?

Some early setups can fail to load dependencies such as `PySide6` or `Pillow` on the first pass. If the editor launch button does not appear, install dependencies again and restart Blender.

## What should I check if the editor button does not appear?

Run the dependency install again and restart Blender once more. That is usually the least interesting failure mode.

## What if the add-on reports missing packages?

Trust the panel instead of guessing. Fix the missing dependencies first, then reload Blender and try again.
