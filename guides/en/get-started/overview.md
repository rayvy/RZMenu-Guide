# RZMenu Guide

Welcome to the RZMenu guide: a practical manual for a tool that is powerful, customizable, and absolutely capable of eating a green beginner alive.

The site is split into four tabs:

- **Get Started**: setup, basic workflow, first successful export.
- **Code Examples**: small copyable patterns and module examples.
- **Assets**: themes, samples, `.rmzct`, `.rzm`, previews, and other useful files.
- **Support Me**: Rayvich links and support notes.

Recommended first path:

1. Read **Installation**.
2. Build the **First Menu**.
3. Bind **Toggles and Textures**.
4. Check **FAQ** before assuming the tool is haunted.

## How to write guides

You write guides as plain Markdown files. No HTML skills required for normal text.

```md
# Big title

Regular text.

## Section title

- List item
- Another list item

![Screenshot](guide-media/example.png)
```

## Where learning images go

Put screenshots, gifs, and short videos used by tutorials into:

```text
guide-media/
```

Keep the visual site theme files in:

```text
assets/
```

Keep downloadable user-shared stuff in:

```text
asset-dump/files/
```
