# ДАВАЙ ЛАБАЙ ДАНЯ

Это стартовая страница RZMenu Guide: мануал для инструмента, который выглядит дружелюбно ровно до того момента, пока новичок не нажал не ту кнопку.

Верхние вкладки:

- **ДАВАЙ ЛАБАЙ ДАНЯ**: установка, старт, первый рабочий экспорт.
- **Код-Залупы**: мелкие примеры, сниппеты, куски модулей.
- **Файлопомойка**: темы, примеры, `.rmzct`, `.rzm`, превьюшки и прочие полезные штуки.
- **ЕслиНеЖалко**: поддержка автора.

Нормальный маршрут для живого человека:

1. Читаешь **Установка**.
2. Собираешь **Первая менюшка**.
3. Привязываешь **Тогглы и текстурки**.
4. Перед истерикой открываешь **ФАКЬЮ**.

## Как писать гайды

Обычными Markdown-файлами. Вёрстку трогать не надо.

```md
# Большой заголовок

Обычный текст.

## Раздел

- Пункт списка
- Ещё пункт

![Скриншот](guide-media/example.png)
```

## Куда кидать картинки для обучения

Скриншоты, гифки и видео для туториалов:

```text
guide-media/
```

Картинки оформления сайта:

```text
assets/
```

Скачиваемые файлы для пользователей:

```text
asset-dump/files/
```
