# Установка

RZMenu живёт в `.zip` архиве из главного GitHub-репозитория:

[rayvy/RZMenu](https://github.com/rayvy/RZMenu)

На GameBanana можно посмотреть, но за актуальными апдейтами туда лучше не ходить. Живые сборки сейчас на GitHub.

## Что делать сначала

1. Скачай `.zip` архив с GitHub.
2. Поставь или обнови аддон в Blender.
3. Открой настройки аддона.
4. Нажми **Install All Missing** в панели зависимостей.
5. Перезапусти Blender после установки пакетов.

Если кнопка редактора не появилась, повтори установку зависимостей и ещё раз перезапусти Blender. Это обычно самый безобидный вид поломки.

## Полезно, но не обязательно

- `RZ Construct Debug` существует для мазохистов.
- Если панель орёт про missing packages, лучше верить панели, а не гадать на кофейной гуще.
- После обновления аддона Blender лучше перезапустить сразу, а не потом удивляться.
