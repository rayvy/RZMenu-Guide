# Установка

Это заготовка. Потом сюда можно вставить твой реальный текст из Discord, только уже нормально разложенный по шагам.

## 1. Ставим нужный Blender

Используй версию Blender, под которую собран текущий RZMenu. Если тулза рассчитана на Blender 5.0, начинай с Blender 5.0.

## 2. Кидаем RZMenu

Папка аддона должна лежать примерно тут:

```text
Blender Foundation/Blender/5.0/scripts/addons/RZMenu
```

## 3. Включаем аддон

Открываешь Blender, идёшь в Add-ons, ищешь RZMenu и включаешь.

## 4. Делаем маленький тест

Не надо сразу тащить огромную модель и страдать. Сначала импортируй маленький тестовый ассет и сделай один экспорт. Цель первого запуска: доказать, что пайплайн вообще живой.
