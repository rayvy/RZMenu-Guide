# Файлопомойка

Да, репозиторий можно использовать как аккуратную файлопомойку. Главное не кидать всё в одну кучу без описаний.

Сюда можно складывать:

- `.rmzct`
- `.rzm`
- примеры модулей
- темы
- тестовые сцены
- скриншоты и превью

Карточки ниже строятся из файла:

```text
asset-dump/catalog.json
```

Сами файлы лежат тут:

```text
asset-dump/files/
```

Превью лежат тут:

```text
asset-dump/previews/
```
