# Код-Залупы

Сюда складываются маленькие примеры, паттерны модулей, конфиги и куски, которые можно скопировать без ритуального чтения исходников в полночь.

## Формат примера

```py
def example_module_entry(context):
    print("RZMenu module example")
```

## Хорошая структура страницы

- Какую боль решает пример.
- Какие файлы трогает.
- Полный сниппет.
- Какой результат ожидать.
- Как это обычно ломают.

> Огромные скрипты лучше класть в файлопомойку, а не прямо в гайд.
