# 开关与贴图

开关是控制网格可见状态最常见的方式。贴图槽则让你把自定义资源绑定到指定网格上。

## 1. 创建一个开关变量

打开 Variables 面板，创建一个 **Toggles (@)** 变量。设置长度，并在需要时设定起始索引。

当你想要离散状态时，就用 `@` 变量：服装变体、可见性状态、菜单页，或者任何像选择器一样工作的东西。

## 2. 把开关分配给网格

在 Blender 里选中网格，打开 RZMenu 的 N 面板，然后使用 **RZ-Toggles -> Assign**。

![Toggle assign panel](guide-media/toggle-assign-panel.png)

从可用的开关里选一个。RZMenu 会给被选中的网格显示一个 bitmask。

## 3. 读懂 bitmask

bitmask 决定这个网格什么时候可见。

例如：

- 如果开关值是 `0`，并且第一个 bit 被启用，那么在状态 `0` 下网格可见；
- 如果开关值是 `1`，并且第二个 bit 被禁用，那么在状态 `1` 下网格隐藏。

这对服装变体、可选部件、替换配件，以及任何需要条件可见性的网格都很有用。

## 4. 分配贴图槽

选中一个网格，然后在 **RZ-TexSlots** 里填写贴图槽名称。

![Texture slot panel](guide-media/texture-slot-panel.png)

如果贴图还不存在，RZMenu 可以为它创建一个槽位。如果它已经在 TexWorks 里存在，也可以从那里拉过来。

## 5. 先把事情做得无聊一点

不要一上来就塞十个条件和五层贴图覆盖。先确认一个开关能控制一个网格的显隐，然后再加复杂度。
