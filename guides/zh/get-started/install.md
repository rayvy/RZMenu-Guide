# 安装

RZMenu 作为 ZIP 压缩包发布，来源是主 GitHub 仓库：

[rayvy/RZMenu](https://github.com/rayvy/RZMenu)

不要指望 GameBanana 上的页面会给你最新版本。它确实存在，但当前构建都在 GitHub 上。

## 最先要做的事

1. 从 GitHub 下载 `.zip` 压缩包。
2. 在 Blender 里安装或更新插件。
3. 打开插件首选项。
4. 在依赖面板里点 **Install All Missing**。
5. 包装依赖装完后，重启 Blender。

如果编辑器按钮没出现，再跑一次依赖安装，然后再重启一次 Blender。通常这就是最不麻烦的失败方式。

## 可选但有用

- `RZ Construct Debug` 是给想主动受苦的人准备的。
- 如果插件提示缺少包，相信面板，不要靠猜。
- 如果你更新了插件，先让 Blender 干净重启一次，再判断结果。
